# README

## 開発言語
- Ruby 3.0.1
- Rails 6.0.3

## 就業Termの技術
- devise
- Ajaxを使ったコメント機能

## カリキュラム外の技術
- gem/rails_admin,gem/cancancaを使った管理者機能
- gem/SimpleCalendar

## 実行手順
````
$ git clone https://github.com/ayaka2209/CHILMO.git
$ cd CHILMO
$ bundle install
$ rails db:create && rails db:migrate
$ rails db:seed
$ rails s
````
