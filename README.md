# README
## 開発言語
* Ruby 3.0.1
* Ruby on Rails 6.0.3

## 就業Termの技術
* Devise
* Ajax

## カリキュラム外の技術
* gem rails_admin, gem cancancanを使った管理者機能
* SimpleCalendar

## 実行手順

```
$ git clone git@github.com:ayaka2209/CHILMO.git 
$ cd CHILMO
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails s
```
